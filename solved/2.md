# Braces Obfuscation
Step 1st:
```javascript
function count(f) {
    let calls = 0;
    function df() {
        calls += 1;
        return f(...arguments);
    }
    df.getCalls = function() {
        return calls;
    }
    return df;
}
```

Step 2nd:
```javascript
function count(f) {
    let calls = 0;
    let df = (...args) => (calls += 1) && f(...args);
    df.getCalls = () => calls;
    return df;
}
```

Step 3rd:
```javascript
const count = f => (calls => {
    let df = (...args) => ++calls && f(...args);
    df.getCalls = () => calls;
    return df;
})(0);
```

Finally:
```javascript
const count = f => ((calls, df) => 
  (df = (...args) => ++calls && f(...args), df.getCalls = () => calls, df)
)(0);
```
